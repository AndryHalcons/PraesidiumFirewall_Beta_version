# Installation

The public beta repository is:

```bash
git clone https://github.com/AndryHalcons/PraesidiumFirewall_Beta_version.git
cd PraesidiumFirewall_Beta_version/installation
chmod +x installer.sh
sudo ./installer.sh
```

Run the installer only on a dedicated test machine or VM until the beta hardening checklist is complete.

Do not change the separate bpfilter repository URL unless bpfilter itself is migrated.
